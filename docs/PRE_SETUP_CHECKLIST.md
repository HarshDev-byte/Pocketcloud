## üö® Common Mistakes to Avoid

### Hardware Mistakes

- [ ] ‚ùå **Don't use cheap power supplies** (causes random crashes)
- [ ] ‚ùå **Don't use old/slow SD cards** (causes slow performance)
- [ ] ‚ùå **Don't skip the case/cooling** (Pi can overheat)
- [ ] ‚ùå **Don't use USB hubs** (can cause power issues)

### Setup Mistakes

- [ ] ‚ùå **Don't skip system updates** (causes compatibility issues)
- [ ] ‚ùå **Don't use weak passwords** (security risk)
- [ ] ‚ùå **Don't forget to write down passwords** (no recovery possible)
- [ ] ‚ùå **Don't rush through USB drive setup** (will erase data!)

### Network Mistakes

- [ ] ‚ùå **Don't use guest networks** (often have restrictions)
- [ ] ‚ùå **Don't skip Ethernet for setup** (Wi-Fi can be unreliable)
- [ ] ‚ùå **Don't forget firewall settings** (blocks access from other devices)
