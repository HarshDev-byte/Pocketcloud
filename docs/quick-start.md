## Management Commands

```bash

# Stop PocketCloud  
./start-pocketcloud.sh --stop

# Check status
./start-pocketcloud.sh --status

# Get help
./start-pocketcloud.sh --help
```

## Access Your Cloud

Once running, access PocketCloud at:
- **Local**: http://localhost:3000
- **Network**: http://[YOUR_PI_IP]:3000
